# 🌦️ Weather Website

📌 Project Title

Weather Website

A dynamic weather web application built using HTML, CSS, and JavaScript.  
This project fetches real-time weather data using an API and displays weather conditions along with background changes based on weather type.

 ✨ Features

🌍 Search Weather by City  
Users can enter a city name to get real-time weather details.

🌡️ Real-Time Temperature  
Displays current temperature in Celsius.

💧 Humidity Display  
Shows humidity percentage of the selected city.

🌬️ Wind Speed  
Displays wind speed information.

🖼️ Dynamic Background  
Background image changes automatically based on weather conditions such as Sunny, Rain, Clouds, or Snow.

🌤️ Weather Icons  
Weather icons update dynamically based on the weather condition.

🎨 Responsive UI Design  
Modern and visually appealing user interface.

 🛠️ Technologies Used

HTML5 – Structure of the website  
CSS3 – Styling and responsive layout  
JavaScript – Weather logic and API integration  
OpenWeather API – Fetch real-time weather data  

🚀 Future Improvements
